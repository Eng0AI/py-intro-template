# AI Agent Instructions for Python Intro Template

This file provides guidance for AI coding agents customizing this Vue.js presentation template for Python tutorials.

## Quick Start

1. **Content**: Edit slide content in Vue components
2. **Code Examples**: Update Python code examples
3. **Styling**: Customize presentation theme

---

## Files to MODIFY (Customize These)

### Content
| Location | Purpose |
|----------|---------|
| `src/` | Slide components with Python examples |

### Assets
| Location | Purpose |
|----------|---------|
| `public/` | Static assets |

---

## Code Examples

When updating Python code examples, ensure:
- Code is properly syntax highlighted
- Examples are correct and runnable
- Comments explain key concepts

---

## Build and Test

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```

---

## Deployment Notes

**Build output**: `dist/` folder

Deploy to any static hosting service.
