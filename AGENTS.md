# Python Intro Template

A Slidev presentation for teaching Python fundamentals - "Python Zero To Hero Episode 1". Features interactive Python code execution within slides.

## Tech Stack

- **Framework**: Slidev v0.51
- **Language**: Vue 3 / Markdown
- **Package Manager**: npm
- **Build Output**: `dist`
- **Special Addons**: Python Runner, Rabbit

## Available Skills

| Skill | Description |
|-------|-------------|
| `build-and-deploy` | Build and deploy the presentation to Vercel or Netlify |

## Quick Start

```bash
npm install
npm run dev
```

## Special Features

- **Interactive Python**: Run Python code directly in slides using `slidev-addon-python-runner`
- **Multiple Themes**: Includes apple-basic, bricks, and penguin themes
- **Progress Tracking**: Rabbit addon for presentation progress

## Customization

- Edit `slides.md` to modify the presentation content
- Slidev uses Markdown with Vue components for slides
- See [Slidev documentation](https://sli.dev) for more features
